# Fullstack Engineer Knowledge Base

## Full Stack Development

### JavaScript/TypeScript Fundamentals
- Async programming: Promises, async/await
- Event-driven architecture
- Module systems: CommonJS, ES6 modules
- Design patterns: Observer, Factory, Singleton
- Functional programming: Map, filter, reduce
- Object-oriented JavaScript: Prototypes, classes
- Testing frameworks: Jest, Mocha, Vitest

### Backend with Node.js
- Express.js framework: Routing, middleware
- Request/response cycle: Headers, status codes
- Error handling: Try-catch, custom error classes
- Authentication: JWT, sessions, OAuth
- Authorization: Role-based access control
- Validation: Input sanitization, schema validation
- Middleware patterns: Logging, compression, CORS

### Frontend with React
- Component lifecycle: Hooks, state, effects
- Virtual DOM: Reconciliation algorithm
- Context API: State management
- Form handling: Controlled components
- API integration: Fetch, Axios
- Performance optimization: Memoization
- Routing: React Router, Next.js

### Databases
- SQL fundamentals: SELECT, INSERT, UPDATE, DELETE
- Indexes: B-tree, hash indexes
- Relationships: One-to-many, many-to-many
- Transactions: ACID properties
- Query optimization: Execution plans
- ORM usage: Sequelize, Prisma, TypeORM
- Database design: Normalization, ER diagrams

### System Design
- Scalability patterns: Horizontal, vertical scaling
- Load balancing: Round-robin, least-connection
- Caching strategies: Redis, memcached
- Message queues: RabbitMQ, Apache Kafka
- Microservices: Service communication
- API design: REST, GraphQL
- Rate limiting: Token bucket, sliding window

### DevOps and Deployment
- Docker: Images, containers, volumes
- Kubernetes basics: Pods, services, deployments
- CI/CD: GitHub Actions, GitLab CI, Jenkins
- Environment management: Dev, staging, production
- Monitoring: Prometheus, Grafana
- Logging: ELK stack, structured logging
- Infrastructure as code: Terraform, CloudFormation

### Security
- HTTPS/TLS: Certificate management
- Authentication: Multi-factor authentication
- Data protection: Encryption, hashing
- Input validation: Preventing injections
- CORS: Cross-origin security
- Dependency security: Vulnerability scanning
- Secret management: Environment variables, vaults
